Set Up Firebase

Community uses Firebase Realtime Database for real friend requests and live chat. It's free and takes about 2 minutes.

1
Go to console.firebase.google.com → create a new project
2
Build → Realtime Database → Create database → choose test mode
3
Project Settings → Your apps → Add web app → copy the config object
4
In this file's <script>, find FIREBASE_CONFIG and paste in your apiKey, databaseURL, and projectId
